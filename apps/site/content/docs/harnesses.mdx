A harness is the headless agent process that powers the chat. Claude is the default.

## Select a harness

```ts title="vite.config.ts"
aidx({
  harness: 'codex',
  harnessBin: '/usr/local/bin/codex',
})
```

`harness` picks the adapter. `harnessBin` overrides the binary on your `PATH`. See
[Configuration](/docs/configuration).

## Available

### Claude

Default. Full support. Needs the `claude` CLI on your `PATH`.

```ts
aidx({ harness: 'claude' })
```

### Codex

Full support.

```ts
aidx({ harness: 'codex' })
```
