---
title: Rspack
description: Add conciv to an Rspack app.
---

Rspack works like webpack. The entry boots the engine, and you inject the widget yourself with
`widgetUrl`.

<Steps>
<Step>

### Install

```bash
npm install -D conciv
```

</Step>
<Step>

### Add the plugin

```js title="rspack.config.js"
const conciv = require('conciv/plugin/rspack')

module.exports = {
  plugins: [conciv.default()],
}
```

</Step>
<Step>

### Inject the widget

The widget bundle ships as `@conciv/widget/global`. Serve it and set `widgetUrl` so the script is
added to your page. See [Configuration](/docs/configuration#widgeturl).

</Step>
</Steps>

<Callout>
  Rspack is less complete than [Vite](/docs/quick-start/vite). The live page bridge is Vite only. Chat, tests, and
  approvals work once the engine is booted and the widget is on the page.
</Callout>
