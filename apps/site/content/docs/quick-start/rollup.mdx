<Callout type="warn">
  The Rollup entry exists but is a build-only no-op today. It does not boot the engine or inject
  the widget yet.
</Callout>

Use [Vite](/docs/quick-start/vite) for the full experience. Vite uses Rollup under the hood, so
most Rollup apps can adopt the Vite plugin directly.
