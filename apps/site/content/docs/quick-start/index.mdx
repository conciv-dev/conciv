## Prerequisites

<Callout>
  You need a dev build of your app, Node 22 or newer, and the `claude` CLI on your `PATH`. The chat uses `claude` to
  answer. See [Harnesses](/docs/harnesses) to use a different agent.
</Callout>

mandarax mounts in dev only. The ✦ button shows once the engine routes are live, so it stays inert on
a plain production preview.
