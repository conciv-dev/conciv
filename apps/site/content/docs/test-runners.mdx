A test runner drives your tests out of process and streams results into the
[test card](/docs/usage/live-tests). Vitest is the default.

## Select a runner

```ts title="vite.config.ts"
aidx({ testRunner: 'playwright' })
```

## Available

### Vitest

Default. Full support.

```ts
aidx({ testRunner: 'vitest' })
```

### Playwright

Full support. Runs `playwright test` and maps the JSON report to the card.

```ts
aidx({ testRunner: 'playwright' })
```
