---
title: What is aidx
description: An embeddable AI dev agent for your running app. Chat, page control, and live tests, injected via a build plugin.
icon: Rocket
---

aidx puts an AI dev agent inside the app you are already running. Add one plugin, and a ✦ button
appears in your dev build. Open it to chat with an agent that sees the page you are on, drives and
edits it, and runs your tests, without leaving the app.

```ts title="vite.config.ts"
import {defineConfig} from 'vite'
import aidx from '@opendui/aidx/plugin/vite'

export default defineConfig({
  plugins: [aidx()],
})
```
